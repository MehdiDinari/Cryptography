# 🔐 Encryption Application (Tkinter GUI)
**A GUI-based encryption and decryption tool using Caesar, Vigenère, and Vernam ciphers.**

## 📌 Features
✅ **Encryption & Decryption:**
   - **Caesar Cipher**: Encrypt and decrypt messages using a shift key.
   - **Vigenère Cipher**: Uses a keyword to encrypt messages.
   - **Vernam Cipher (One-Time Pad)**: A perfect encryption method using a key of the same length as the message.

✅ **Brute-Force Attack:**
   - **Caesar Cipher**: Automatically tries all possible shift values (`k = 1 to 25`) to recover the original message.

✅ **Graphical User Interface (Tkinter):**
   - User-friendly interface.
   - Error handling and input validation.
